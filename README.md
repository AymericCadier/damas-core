# damas-core

Key value storage system with graph capabilities

* HTTP Php web service
* Javascript bindings
* Python bindings
* MySQL database backend

## javascript
```html
<script type="text/javascript" src="js/damas.js"></script>
```

## python
```python
import('damas.py')
```
